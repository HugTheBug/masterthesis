\chapter*{Заключение}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

	В данной дипломной работе была рассмотрена задача нахождения системы методов для оптимального управления  беспилотным автомобиля по шоссе. 
	Для исследуемой задачи был сформулирован метод нахождения оптимального управления движением, обеспечивающий поддержание постоянной сокорости движения для электромобиля и корректного прохождения поворотов. Результаты применения данного метода, а также подтверждение его работоспособности проиллюстрированы численными экспериментами для трассы близкой к реальной двухполосной дороге.
	Так же был определен метод для построения оптимальных траекторий движения автомобиля по многополосному шоссе в условиях потока машин. Результаты применения данного метода в комбинации с выше упомянутым методом управления движения продемонстрированы численными экспериментами для трассы близкой к реальному трехполосному шоссе. 
	Во всех численных экспериментов в качестве модели автомобиля была взята Tesla Model 3 Standard Range Plus, хотя оба метод были разработан с учетом того, что параметры автомобиля могут отличаться.
	 
	Для дальнейшего развития метода стоит улучшить постороение траекторий движения автомобиля, что должно исправить скачки в графиках отклонений и улучшить проведение маневров, особенно для более высоких чем текущий диапазонов скоростей. Так же для корректности управления при больних скоростях следует использовать динамическую модель для MPC вместо кинетической. Еще одним улучшением для повышения комфорта пассажиров будет являться автоматическое понижение желаемой скорости в зависимости от кривизны дороги. Для улучшения точности поддержания скорости следует провести автоматическую вместо ручной настройку параметров для PID, к примеру методом Зиглера-Никольса. Другим способом улучшения контроля скорости будет использование одного из похожих на PID регуляторов специально разработанных для этой задачи.
	
	Для более точного определения эффективности приведенного метода следует провести численные эксперименты в условиях городской среды при прохождении перекрестков, каруселей и других сложных участков дороги.
	
	Изложенная система методов не предназначена для решения задач автоматической парковки, что опять таки может быть улучшено при дальнейшем развитии.
