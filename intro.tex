\chapter*{Введение}
\addcontentsline {toc}{chapter}{ВВЕДЕНИЕ}

	В данной работе рассматривается проблема нахождения подхода для определения оптимального управления  бесилотным автомобилем по многополосному шоссе. Для определения оптимальности управления автомобилем учитывается отклонения центра и угла поворота машины от центра выбранной полосы, отклонения от заданной скорости движения, гладкость полученного управления. 

	Атуальность задачи связана с развитием современного автомобилестроения и необходимостью разработки методов управления беспилотными автомобилями.
	
	Целью работы является создание системы методов предназначенных для управления беспилотными автомобилями и решающих задачи построения траекторий и управления движением.

	В первой главе рассматривается общая теория для MPC методов и приводятся основные причины их использования, также приведена информация о библиотеке для реализации данных методов. Наиболее важной частью первой главы является MPC метод для следования пути, который будет применяется для управления автомобилем. Во второй главе находится краткое описание PID методов.  В третьей главе приводится описание непосредственно задачи, решаемой в данной работе. Также в третьей главе приводится детальное описание метода решения данной задачи и симулятора беспилотных автомобилей, в котором будут проводится эксперименты.

	В заключении приводятся краткие выводы по результатам проведенной работы и даются рекомендации о перспективах дальнейших исследований по исследуемой тематике.
