\chapter*{Введение}
\addcontentsline {toc}{chapter}{ВВЕДЕНИЕ}

	В данной работе рассматривается проблема нахождения подхода для определения оптимального управления  беспилотным автомобилем по многополосному шоссе. Для определения оптимальности управления автомобилем учитывается отклонения центра и угла поворота машины от центра выбранной полосы, отклонения от заданной скорости движения, гладкость полученного управления. 

	Актуальность задачи связана с развитием современного автомобилестроения и необходимостью разработки методов управления беспилотными автомобилями.
	
	Целью работы является создание системы методов, предназначенных для управления беспилотными автомобилями и решающих задачи построения траекторий и управления движением.

	В первой главе рассматривается общая теория для MPC, PID методов и приводятся основные причины их использования, также приведена информация о библиотеке для реализации данных методов и информация о методе для построения траекторий. Наиболее важной частью первой главы является описание NMPC методов и метода построения траекторий. Во второй главе приводится описание первой задачи, решаемой в данной работе, связанной со следованием автомобилем заранее построенной траектории. Также во второй главе приводится детальное описание метода решения данной задачи и симулятора беспилотных автомобилей, в котором были проведены эксперименты. В третьей главе приводится описание задачи построения траекторий движения беспилотного автомобиля по шоссе.

	В заключении приводятся краткие выводы по результатам проведенной работы и даются рекомендации о перспективах дальнейших исследований по исследуемой тематике.
